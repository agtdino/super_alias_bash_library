# super_alias_bash_library
# agtdino / Mondrillo author

common bash aliases for it administrators

Clone git repository into ~/ directory:

-) cd ~/
-) git clone https://github.com/agtdino/super_alias_bash_library.git

Include super_alias_bash_library editing ~.bashrc:

-) echo "source ~/super_alias_bash_library/super_alias_bash_library">> ~/.bashrc

Update actual .bashrc configuration:

-) source .bashrc

Now you can run super alias in your bash sessions.
